Catholic Social Services Online Assessment Tool for Caregivers in Washtenaw County

This is an infoLab project working with Catholic Social Services building up an online assessment tool for caregivers who take care of older adults.

Our project is a small part of a larger picture. The tool we are working on is going to return the directory based on another student team's research result. What we focus on is studying how users access to the resource and what the structure of the tool should be.

  • User Experience Design
  • User Research
  • Interface Design
  • Competitive Analysis
