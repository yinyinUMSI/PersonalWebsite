Urgent and Important Matrix Reminder

This is an individual project primarily for practicing Vanilla JS before stepping into jQuery and using front-end frameworks. I used indexedDB to store the data. Moreover, I used OOP throughout the whole project, which gives me a better understanding of the objects in JS.

Please visit https://yinyinumsi.github.io/UrgentImportantMatrixOnline/ to have a try. The code is on Github Urgent&ImportantMatrixOnline.

  • CSS

Functions

Add a note

  • Title, deadline and type are required fields. If not completed, the browser will remind the user to finish all. If user try to close the form by clicking the pink button, it will alert user that the record will not be saved.
  • After clicking the complete button, a new reminder will be added and recorded to the indexedDB.

    • Delete a note

  • After finishing a task, user can click the checkmark on the left top, which will trigger the deletion of that record in the DB and close the reminder.

    • Things to improve

  • Delete the database in indexedDB when the user open the app, so that there won’t be reluctant data left by the added but not deleted records.
  • ndexedDB can be manipulated by Promise. I should have a try later on.
  • Allow user to drag a note from one area and drop in another. This is a great challenge to work on.
